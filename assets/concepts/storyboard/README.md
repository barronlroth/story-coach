# Story Coach UX Storyboard

These mockups use the horizontal drawing-reference style from
`story-coach-main-character-description-horizontal.png`.

## Flow

1. `00-intro-start.png` - Intro/start screen
2. `01-step-1a-draw-main-character.png` - Draw main character
3. `02-step-1b-describe-main-character.png` - Describe main character
4. `03-step-1c-generating-character.png` - Generate character
5. `04-step-1d-confirm-character.png` - Confirm character
6. `05-step-2a-draw-special.png` - Draw what makes them special
7. `06-step-2b-describe-special.png` - Describe what makes them special
8. `07-step-2c-confirm-special.png` - Confirm special trait/detail
9. `08-step-3a-describe-want.png` - Describe what they want
10. `09-step-3b-confirm-want.png` - Confirm goal
11. `10-step-4a-draw-problem.png` - Draw the problem
12. `11-step-4b-describe-problem.png` - Describe the problem
13. `12-step-4c-confirm-problem.png` - Confirm problem
14. `13-step-5a-draw-try.png` - Draw what they try
15. `14-step-5b-describe-try.png` - Describe what they try
16. `15-step-5c-confirm-try.png` - Confirm attempt
17. `16-step-6a-draw-ending.png` - Draw the ending
18. `17-step-6b-describe-ending.png` - Describe the ending
19. `18-step-6c-confirm-ending.png` - Confirm ending
20. `19-building-book.png` - Assemble the finished book
21. `20-finished-book-reader.png` - Finished book reader

## Reusable States

- `21-add-detail-correction.png` - Add a spoken correction/detail
- `22-recording-active.png` - Active voice recording
- `23-gentle-retry-error.png` - Gentle retry/error state
- `24-book-edit-words.png` - Light editing for finished book text
